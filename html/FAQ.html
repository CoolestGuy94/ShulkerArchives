Frequently Asked Questions

Is there a Discord?

You can join our Discord Server here! We post regular updates about the Archives there, as well as dicuss a lot of NBT/Archhive related things.

Using .nbt files?

Check out the Using NBT page for instructions on using .nbt files, and other related information.

How can I submit a Kit or Item?

Yes! The best way to contact is via our Discord, or by DMing Kaddicus#2019

Who are the Archivists?

There are 3 primary archivists: KaddicusAlmighty, _win9x_ and Parsa__, although many others like CoolestGuy94, Emphacks, Yurni and FileExtension have contributed in different ways :)

Can I help the Archives?

Absolutely! We don't currently have a uniform list of things we need help with, so contact Kaddicus#2019 on Discord to see about what you can help with.

I'm interested in NBT, where should I start?

A good place to start is our Resources and Guides pages, where we have a lot of information for all things NBT Related.

Welcome to the Shulker Archives

Below is a list of resources commonly used in making Shulker Kits

Meteor Client
A Mod/"Client" for Fabric that has excellent commands for altering NBT, among many other useful nodes.

MCStacker
One of the best resources for creating Custom Items and altering NBT. If you're new to Item Creation and NBT, MCS is a great place to start.

Minecraft-ArmorStand
Allows you to create custom armor stands, with a visual model of the stand itself to help you.

Colorize.fun
Helps in colorizing your Minecraft text with solid colors and gradients

Minecraft Chat Symbols
Has a list of symbols, emoticons and characters supported by Minecraft

NBT Studio
NBT Studio is an NBT editing application, the spiritual successor to NBTExplorer.

MapartCraft
Lets you upload an image, and converts it into a Minecraft Map. You can customize Blocks, Dithering, Staircasing, etc, and then download either the resulting image, a Schematic file, or a map data file.

R74n
An interesting amalgamation of resources and tools for Minecraft and other things. Of particular note is the hidden Box Page containing the /give commands and NBT for a number of Shulker Kits, including Ryan UwU 1-7.
